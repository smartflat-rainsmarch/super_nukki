# CLAUDE.md

## 목적
이 프로젝트는 UI 이미지를 분석하여 PSD 파일로 변환하는 SaaS이다.

## 핵심 규칙
1. 항상 MVP부터 구현한다
2. 한 번에 하나의 기능만 개발한다
3. 변경 파일 목록을 먼저 제시한다
4. 실행 방법과 테스트 방법을 반드시 포함한다
5. 문서(PRD 등)와 충돌 시 수정안 제시
6. 과도한 추상화 금지
7. 다음 진행 요청 시 자동으로 다음 단계 수행

## 개발 순서
1. 프로젝트 초기 세팅
2. 업로드 기능
3. 분석 파이프라인
4. OCR
5. 요소 분리
6. 배경 복원
7. PSD 생성
8. 상태 관리
9. 결제 시스템
