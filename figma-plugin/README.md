# UI2PSD Figma Plugin

UI2PSD Studio에서 변환한 레이어 데이터를 Figma로 import하는 Plugin입니다.

## 설치 방법 (로컬 개발)

1. Figma Desktop 앱 → Plugins → Development → Import plugin from manifest
2. `figma-plugin/manifest.json` 파일 선택
3. Plugin 실행: Plugins → Development → UI2PSD Import

## 사용법

1. UI2PSD에서 "Figma로 보내기" → Share Code 복사
2. Figma에서 Plugin 실행 → Share Code 입력
3. "Import to Figma" 클릭
4. 레이어가 자동으로 Figma Canvas에 생성됩니다
