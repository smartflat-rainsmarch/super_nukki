# UI Image → PSD SaaS PRD

## 1. 제품 개요
이미지(UI 스크린샷)를 업로드하면
자동으로 요소를 분석하여 PSD 파일로 변환하는 SaaS

## 2. 핵심 가치
- 디자이너: 재작업 시간 절감
- 개발자: 리소스 추출 자동화
- AI 생성 이미지 → 실무 사용 가능

## 3. 주요 기능

### 3.1 업로드
- PNG, JPG 지원
- 최대 4096x4096

### 3.2 분석
- UI 컴포넌트 분리
  - 버튼
  - 텍스트
  - 아이콘
  - 배경

### 3.3 OCR
- 텍스트 추출 및 폰트 유추

### 3.4 이미지 분할
- 각 요소 PNG로 추출

### 3.5 배경 복원
- 인페인팅으로 빈 영역 채움

### 3.6 PSD 생성
- 레이어 구조 포함
- 그룹 구조 자동 생성

## 4. 사용자 플로우

1. 로그인
2. 이미지 업로드
3. 분석 (10~30초)
4. 결과 미리보기
5. PSD 다운로드

## 5. 요금제

### 무료
- 월 3회
- 해상도 제한 1024px
- 워터마크 포함

### 일반 ($39/month)
- 월 100회
- HD 지원
- 워터마크 제거

### 프로 ($129/month)
- 무제한
- 고속 처리
- API 제공
- PSD 고급 옵션

## 6. KPI
- 변환 성공률 90%
- 평균 처리시간 20초 이하
- 유료 전환율 5%+
